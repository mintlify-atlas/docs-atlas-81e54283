# Welcome to Home Manager

Home Manager is a comprehensive household management platform that helps families and roommates organize their daily lives. Track chores, manage bills, create shopping lists, log maintenance tasks, and keep everyone in your household on the same page.

## Built With Modern Technology

Home Manager is built on a powerful tech stack designed for reliability and scalability:

- **Next.js 15** - React framework with App Router
- **TypeScript** - Type-safe development
- **Prisma ORM** - Type-safe database access
- **PostgreSQL** - Robust relational database
- **Clerk** - Authentication and user management
- **Material-UI** - Beautiful, accessible components

<Note>
  Home Manager supports role-based access control with Owner, Member, and Guest roles to ensure proper permissions across your household.
</Note>
