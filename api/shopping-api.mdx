## Overview

The Shopping API allows you to create, read, update, delete, and reorder shopping list items. Each item includes a name and category for organization.

## Endpoints

### Get All Shopping Items

<ParamField path="GET /api/shopping" type="endpoint">
  Retrieves all shopping items for the authenticated user's household, ordered by position.
</ParamField>

**Authentication**: Required

**Response**

<ResponseField name="items" type="array">
  Array of shopping item objects
</ResponseField>

<Expandable title="Shopping Item Fields">
  <ResponseField name="id" type="string">
    Unique identifier
  </ResponseField>
  
  <ResponseField name="householdId" type="string">
    ID of the household this item belongs to
  </ResponseField>
  
  <ResponseField name="name" type="string">
    Name of the item (e.g., "Milk", "Bread")
  </ResponseField>
  
  <ResponseField name="category" type="string">
    Item category (e.g., "Dairy", "Produce", "Pantry")
  </ResponseField>
  
  <ResponseField name="position" type="number">
    Position for ordering in the list
  </ResponseField>
  
  <ResponseField name="checked" type="boolean">
    Whether the item has been purchased (default: false)
  </ResponseField>
  
  <ResponseField name="createdAt" type="datetime">
    Creation timestamp
  </ResponseField>
</Expandable>

<CodeGroup>
```bash cURL
curl -X GET https://your-domain.com/api/shopping \
  -H "Authorization: Bearer YOUR_CLERK_TOKEN"
```

```javascript JavaScript
const response = await fetch('/api/shopping');
const items = await response.json();
```
</CodeGroup>

**Example Response**

```json
[
  {
    "id": "clx123abc",
    "householdId": "clx456def",
    "name": "Milk",
    "category": "Dairy",
    "position": 0,

    "createdAt": "2024-03-15T10:00:00Z"
  },
  {
    "id": "clx789ghi",
    "householdId": "clx456def",
    "name": "Bananas",
    "category": "Produce",
    "position": 1,
    "checked": false,
    "createdAt": "2024-03-15T11:00:00Z"
  }
]
```

### Create or Reorder Shopping Items

<ParamField path="POST /api/shopping" type="endpoint">
  Creates a new shopping item or reorders existing items.
</ParamField>

**Authentication**: Required

**Request Body (Create)**

<ParamField name="name" type="string" required>
  Name of the item
</ParamField>

<ParamField name="category" type="string" required>
  Item category
</ParamField>

**Request Body (Reorder)**

<ParamField name="orderedIds" type="array" required>
  Array of item IDs in the desired order
</ParamField>

<CodeGroup>
```bash Create Item
curl -X POST https://your-domain.com/api/shopping \
  -H "Authorization: Bearer YOUR_CLERK_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Eggs",
    "category": "Dairy"
  }'
```

```bash Reorder Items
curl -X POST https://your-domain.com/api/shopping \
  -H "Authorization: Bearer YOUR_CLERK_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "orderedIds": ["clx789ghi", "clx123abc", "clxnew123"]
  }'
```

```javascript Create Item
const response = await fetch('/api/shopping', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    name: 'Eggs',
    category: 'Dairy',
  }),
});

const newItem = await response.json();
```
</CodeGroup>

**Response (Create)**

```json
{
  "id": "clxnew123",
  "householdId": "clx456def",
  "name": "Eggs",
  "category": "Dairy",
  "position": 2,
  "checked": false,
  "createdAt": "2024-03-15T12:00:00Z"
}
```

**Response (Reorder)**

```
Positions updated
```

Status: `200 OK`

<CodeGroup>

### Delete Shopping Item

<ParamField path="POST /api/shopping/delete" type="endpoint">
  Deletes (marks as purchased) a shopping item and creates a notification.
</ParamField>

**Authentication**: Required

**Request Body**

<ParamField name="id" type="string" required>
  ID of the item to delete
</ParamField>

<CodeGroup>
```bash cURL
curl -X POST https://your-domain.com/api/shopping/delete \
  -H "Authorization: Bearer YOUR_CLERK_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"id": "clx123abc"}'
```

```javascript JavaScript
const response = await fetch('/api/shopping/delete', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ id: 'clx123abc' }),
});

const deletedItem = await response.json();
```
</CodeGroup>

**Response**

```json
{
  "id": "clx123abc",
  "householdId": "clx456def",
  "name": "Milk",
  "category": "Dairy",
  "position": 0,
  "checked": false,
  "createdAt": "2024-03-15T10:00:00Z"
}
```

<Note>
  When a shopping item is deleted, a notification is automatically created with:
  - **Type**: "shopping"
  - **Title**: "Item Removed"
  - **Body**: "You completed: [item name]"
</Note>

### Get Shopping Items Count

<ParamField path="GET /api/shopping/count" type="endpoint">
  Returns the total number of shopping items for the household.
</ParamField>

**Authentication**: Required

<CodeGroup>
```bash cURL
curl -X GET https://your-domain.com/api/shopping/count \
  -H "Authorization: Bearer YOUR_CLERK_TOKEN"
```

```javascript JavaScript
const response = await fetch('/api/shopping/count');
const { count } = await response.json();
```
</CodeGroup>

**Response**

```json
{
  "count": 12
}
```

<Note>
  The count endpoint returns `{"count": 0}` if the user is unauthorized, rather than throwing an error. This is different from other count endpoints.
</Note>
